# Self-hosted-open-source-LLM
Self-hosted веб-приложение, которое позволяет пользователю ввести описание конечной цели проекта и мгновенно получить готовую иерархическую структуру работ (ИСР) в формате документа или таблицы. Система включает в себя базу знаний с ГОСТами и стандартами, а также понятную админку для её пополнения неподготовленным пользователем.
